# Vehicle Loan Default Prediction

This project aims to predict the likelihood of a customer defaulting on a vehicle loan. It uses machine learning techniques to analyze historical data and provide insights for risk assessment.

## Project Structure

- `data/`: Contains raw, processed, and external datasets.
- `notebooks/`: Jupyter notebooks for EDA and modeling experiments.
- `src/`: Source code for the project.
    - `data/`: Scripts for data processing.
    - `features/`: Logic for feature engineering.
    - `models/`: Model training and evaluation.
    - `visualization/`: Utility functions for plots.
- `models/`: Saved model files.
- `reports/`: Generated reports and figures.

## Getting Started

1.  **Clone the repository.**
2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Explore the data:** Check the `notebooks/` directory for exploratory data analysis.
